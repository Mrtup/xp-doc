Writing libraries
=================

In Enonic XP, there are unlimited ways to extend the functionality.  Libraries are one way to create reusable components and make
them available for use in different apps.  They may or may not be GUI components.

Enonic XP comes with a number of libraries included in the core distribution.  For more about them, check out: :ref:`js-libraries` .
