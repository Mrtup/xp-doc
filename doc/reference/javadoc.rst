JavaDoc API
===========

You can either download the JavaDoc `as a zip`_ or view it
`directly in your browser`_.

.. _as a zip: http://repo.enonic.com/public/com/enonic/xp/docs/${release}/docs-${release}-javadoc.zip
.. _directly in your browser: http://repo.enonic.com/public/com/enonic/xp/docs/${release}/docs-${release}-javadoc.zip!/index.html
